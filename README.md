# MoreOres 更多矿物 [魔改版]

这是一个 [Slimefun4](https://github.com/happy66dev/Slimefun4) 插件附属。

想法提出者: 小斗 | 实现: happy

## 下载

点击这里下载 MoreOres: [下载 MoreOres](https://github.com/happy66dev/MoreOres/releases)

## 内容

为游戏新增 **43 种矿物**，可通过挖掘原版方块、磨石研磨、GEO 矿机三种方式获得。

## 特性

- 所有矿物**不可放置**、**不可在工作台中使用**、**不可在熔炉中烧制**
- 使用**精准采集**附魔的工具挖掘时，矿物**不会额外掉落**
- 所有掉落概率**可在 config.yml 中自定义**
